Virginia Crash Data Analysis

This project is for the Independent Study Project.

What is this project about?

This is an app similar to Google Maps. It shows your current location in a map. It also displays zones. These zones are areas in Virginia where the most number of crashes have happended. When you enter these zones, you will recieve a notification on your device, telling you to be careful and more cautious. This app was made using HTML, CSS, and Javascript. I also used Python to analyze the data and get the coordinates of the crashes to display it in the app.
